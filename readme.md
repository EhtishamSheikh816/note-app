# Note App

A modern Note App built with **React**, **Tailwind CSS**, **Node.js**, **Express**, **MongoDB**, and **JWT** for authentication. This app allows users to securely create, edit, delete, and search notes, with a responsive UI and cookie-based authentication.

---

## Features

- **User Authentication:** Secure login and signup with JWT and cookies.
- **Create, Edit, Delete Notes:** Full CRUD functionality for managing notes.
- **Pin Notes:** Option to pin important notes for easy access.
- **Search Notes:** Quickly find notes using a search bar.
- **Responsive UI:** Works on both desktop and mobile devices.
- **Read More:** Expand long notes in a modal to view full content.
- **Dark/Light Theme Toggle:** Switch between themes for better UX.

---

## Technologies Used

- **Frontend:** React, Tailwind CSS, React Hook Form, React Router
- **Backend:** Node.js, Express
- **Database:** MongoDB, Mongoose
- **Authentication:** JWT, Cookies
- **Other:** Axios, React Hot Toast, Lucide Icons
